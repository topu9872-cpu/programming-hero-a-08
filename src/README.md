## A Online Book Borrowing Platform

A modern, web application designed to manage book borrowing and lending. Built with the latest technologies to ensure speed, security, and a seamless user experience.

## Tech Stack
HTML
Tailwind CSS
JavaScript(ES6)
React
Next.js 
Next.js Server Actions
MongoDB
Better-Auth
Vercel
Dasyui
HeroUi
React Toastfy
React icons
React form
Herogravity
eact-fast-marquee
animate.CSS

    **User Authentication:** Secure login and registration using Better-Auth.

**Book Management:** Browse a collection of books with details like author, genre, and availability.
**Borrowing System:** Users can borrow books and keep track of their active rentals.
**Dashboard:** A personalized dashboard for users to manage their profile and borrowed items.
**Responsive Design:** Fully optimized for mobile, tablet and desktop views.
**Real-time Updates:** Smooth interaction with the database using MongoDB.
